File Guidelines
  • File Format: Only FASTA (.fasta, .fa, .fas, .fna, .ffn, .faa, .frn) or plain text (.txt) files are supported.
  • File Size: Maximum file size is 2GB. Larger files may take longer to process.
  • Sequence Quality: Ensure sequences are of high quality with minimal N characters.
  • Sequence Length: Reference sequences should be at least 1000 bases for best results.
  • Multiple Sequences: Files can contain multiple sequences, but only the first one will be used as the reference.
Naming Convention

The system will try to identify the species based on the filename. Include one of the following keywords in the filename for automatic identification:

human
mouse
zebrafish
arabidopsis
ecoli
bsubtilis
